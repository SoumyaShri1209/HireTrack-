# HireTrack — AI-Powered Smart Job Application Tracker

A full-stack MERN application that helps job seekers manage their entire job search in one place — from resume parsing and multi-source job aggregation to Kanban pipeline tracking and offer comparison.

---

## 📂 Project Structure

| Folder | Description |
|:-------|:------------|
| [hiretrack-backend](./hiretrack-project/hiretrack-backend) | Node.js/Express REST API |
| [hiretrack-frontend](./hiretrack-project/hiretrack-frontend) | React + Vite frontend |

---

## 🛠️ Tech Stack

**Backend:** Node.js, Express.js, MongoDB, JWT, Google OAuth, Gemini AI, Nodemailer

**Frontend:** React, Vite, Tailwind CSS, React Router, Axios

---

## 🚀 Quick Start

1. Clone the repository
2. Set up the [backend](./hiretrack-project/hiretrack-backend)
3. Set up the [frontend](./hiretrack-project/hiretrack-frontend)
4. Visit `http://localhost:5173`
